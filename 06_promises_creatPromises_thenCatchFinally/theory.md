
Promises - это специальный объект, который содержит состояние (pending, fulfilled и rejected) и служит обёрткой для асинхронной операции, позволяя обрабатывать её результат или ошибку в будующем.

Pending - ожидание (начальное состояние)
Fulfilled (resolved) - успешно выполнено
Rejected - выполнено с ошибкой

Основные методы:

1. .then(onFulfilled, onRejected)
    - Главный метод. Принимает два колбека (второй необязательный)
    - Вызывается, когда промис завершился (успех или ошибка)
    - Возвращает новый промис, что позволяет строить цепочки промисов.
2. .catch(onRejected)
    - Специальный метод для отлова ошибок в цепочке промисов
    - Ловит любые ошибки из всех предыдущих .then в цепочке
    - Фактически является методом .then(null, onRejected)
3. .finally(onFinally)
    - Выполняется вне зависимости от результата
    - Не получает аргументов (ни результата, ни ошибку) - только фиксирует факт завершения
    - Идеален для финальных операций
